**Family Budgeting Application Database:**
 A robust and scalable relational database schema designed to empower families with comprehensive financial tracking and management tools.

**Project Overview:**
The goal of this project was to design a structured data environment that allows users to track income, monitor expenses, and manage long-term savings goals within a single, unified system. The architecture prioritizes data integrity and performance to provide clear, real-time financial insights.


**Key Features**
* Comprehensive Tracking: Integrated modules for income, expenses, and personalized savings goals.
* Data Integrity: Implementation of 3rd Normal Form (3NF) and ER diagram modeling to eliminate redundancy and maintain high data standards.
* Optimized Performance: Fine-tuned SQL queries designed to generate financial reports and summaries efficiently.
* Scalable Architecture: Designed with a relational approach to accommodate expanding datasets and complex user structures.

**Tech Stack**
* Language: SQL 
* Systems: PostgreSQL / MS SQL Server (Database Management)
* Methodology: Relational Data Modeling and ER Diagramming 

**Database Schema:**
The schema is built around core financial entities, ensuring a logical flow between users, categories, and transactions.
* Normalization: Rigorous application of normalization techniques to ensure system stability.
* Query Optimization: Custom SQL scripts developed to calculate monthly balances, expense distributions, and progress toward savings targets.

**Implementation Highlights**
* ER Modeling: Detailed mapping of entities and relationships to mirror real-world financial interactions.
* Reporting: Automated queries for instant financial health snapshots.
